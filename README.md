# Expense Tracker App

A simple full-stack expense tracker application built to understand
frontend-backend integration and CRUD operations.

## Features
- Add, edit, and delete expenses
- Filter expenses by category
- Calculate total expenses
- Persistent data storage using MongoDB

## Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB

## How to Run Locally

### Backend
cd Expense-Tracker-backened
npm install
node server.js

### Frontend
cd Expense-Tracker-frontened
npm install
npm run dev
